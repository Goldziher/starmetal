# StarMetal

StarMetal is a high-performance, self-hosted multi-language package registry and pull-through
registry proxy for private software supply chains.

This crates.io package currently reserves the public `starmetal` namespace while the production Rust
crate layout is finalized. Use the Docker image, Homebrew bottle, npm/PyPI binary installer, GitHub
release archive, or workspace build for the current `sm` CLI.

Repository: <https://github.com/Goldziher/starmetal>
