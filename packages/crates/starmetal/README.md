# StarMetal

StarMetal is a multi-language, high-performance, self-hosted package registry and registry proxy.

This `0.0.1` crates.io package reserves the public `starmetal` namespace while the production Rust
crate layout is finalized. Use the Docker image or build the workspace binary from source for the
current server.

Repository: <https://github.com/Goldziher/starmetal>
